# lumen — 빛의 역사 (인터랙티브 실험)

기원전 40만 년의 모닥불에서 지금 이 화면의 픽셀까지, **스크롤 = 시간여행**으로 인류가 밤을 건너온 이야기를 담은 스크롤 시네마 실험 부품.

스토어프론트 적용 가능성 테스트용 오리지널 제작물 (2026-07).

## 실행

정적 서버로 열면 된다 (모듈/빌드 없음, 단일 파일):

```
npx http-server cheese-blog-parts -p 8932
→ http://localhost:8932/lumen/
```

## 구성

- `index.html` 하나가 전부. 외부 라이브러리 0 (폰트만 Google Fonts, 오프라인 시 serif 폴백).
- 순수 Canvas 2D + WebAudio(전부 합성음, 오디오 파일 없음).
- 9개 챕터: 서장 → 모닥불 → 등잔 → 촛불 → 가스등 → 백열전구 → 네온 → 스크린 → 에필로그.
- 스크롤 진행도(0..1)를 챕터별 smoothstep 밴드 가중치로 바꿔 장면을 크로스페이드.

## 상호작용

- 마우스 이동 = 바람(불꽃이 눕는다), 촛불 챕터는 커서 근접 시 불꽃이 흔들림.
- 클릭 = 불씨 버스트 + 화력 증가.
- 우측 레일 도트 = 시대 점프. 좌하단 버튼 = 사운드 토글(기본 무음).
- `prefers-reduced-motion` 지원 (파티클 감소, 셰이크/그레인 제거, 즉시 스크롤).

## 개발 메모

- `?headless` 쿼리: rAF를 setTimeout으로 대체 — 브라우저 패널이 숨겨진 상태(rAF 정지)에서도 픽셀 검증 가능.
- 성능: glow는 128px 스프라이트 캐시(drawImage), 네온 간판 shadowBlur는 오프스크린 1회 렌더 후 합성, '빛' 픽셀 그리드는 채널별 배칭. 프레임당 그라디언트/문자열 생성 없음.
- 검증 이력: 멀티에이전트 리뷰(5차원 × 적대적 검증)로 확정된 22건 수정 완료.
