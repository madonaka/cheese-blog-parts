# 2025-06-09 회의 — 9차 전략 기획: 주간 성과 점검 + UX 실험 리뷰

## 배경 (왜 이 회의를 했나)

한 주(6/3~6/9)의 실행 성과를 점검하고, 그동안 실험한 블로그스팟 콘텐츠 허브 UX·네이버 외부 유입·자동 목차 기능의 1차 결과를 리뷰하는 자리. (콘텐츠 기획 자체는 이번 안건에서 제외.)

---

## 오늘 정한 것 (결론)

1. **주간 실행 성과 100% — 조선왕조 가계도 시리즈 정비 완료.**
   효종~순종까지 매일 1편씩 6편 전량 티스토리 업로드 완료(효종&현종은 1편 통합). 계획 대비 일정 지연·누락 없음. 인덱스 페이지·외부 유입 콘텐츠까지 추가 실험. 이로써 콘텐츠 라인업 확보(중기 ①)의 기반 완성.

2. **블로그스팟 UX — 구조는 완성, 반응은 저조 → 개선 필요 확정.**
   초기-중기-후기 3탭 + 콘텐츠 카드 구조는 구성 완료(완성도 70%). 그러나 실제 탐색/클릭 행동은 저조하고, 페이지 이탈률이 높고 CTA 전환률이 낮음. 개선 요소: 동선 단순화·CTA 위치 재배치·검색 최적화.

3. **네이버 외부 유입 실험이 더 실용적일 가능성 확인.**
   네이버 블로그에 가계도 요약 콘텐츠 5편 게시(CTA "전체 보기/가계도 보러 가기/더 알아보기" 삽입), 글당 평균 10회 내외 조회(최고 14회). **블로그스팟보다 네이버 → 티스토리 직접 유도가 더 실용적**일 수 있다고 판단.

4. **자동 목차(h2 기반)를 탐색 UX 요소로 정식 반영.**
   긴 티스토리 글에 h2 기반 자동 목차 삽입 스크립트 적용. 2~3일 디버깅 후 안정화 완료. 소제목 파악·이동이 쉬워져 체류 시간 향상 기대.

## 아직 안 정한 것 (다음에, 필요해지면)

- **블로그스팟 UX 개선 전략**: 동선 간소화·CTA 재설계 — 다음 회의에서 본격 논의.
- **외부 유입 A/B 테스트 + 티스토리 전환율 정량화**: 현재 전환율 데이터 없음 → GA4 연동 필요.
- **목차 클릭률 추적 여부**: 기능적 확장 가능성 점검.
- **블로그스팟 애드센스 연결**: 광고 위치/조건·승인 구조 정리 예정.
- **도메인 구매(가비아 등)**: 브랜드용 도메인 확보 시점·명칭 전략 미정.
