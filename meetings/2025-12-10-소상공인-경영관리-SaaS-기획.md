# 2025-12-10 회의 — 소상공인 맞춤형 올인원 관리자페이지(경영관리 SaaS) 기획

> 참여: 치즈 (단독, ChatGPT 상담)
> 주제: POS·회계 프로그램 없이 수기·엑셀로 운영하는 소상공인 대상 올인원 경영관리 SaaS 기획 요약
> ※ 이관 노트. Notion 기획 요약 보고서를 회의록 형식으로 압축. **역사 콘텐츠(Luminary)와 별개의 사업 아이디어**라 project는 붙이지 않음. (지금까지 만든 백오피스·CRM 시스템을 제품화하는 구상.)

---

## 배경 (왜 이 회의를 했나)

POS·컴퓨터·회계 프로그램 없이 수기장부·카톡·엑셀로 매장을 운영하는 소상공인(3~10인)을 대상으로, "엑셀 지옥에서 벗어나게 해주는 간단한 자동화"를 파는 올인원 경영관리 SaaS를 구상했다. 그동안 만든 백오피스·CRM·예약·급여 시스템을 제품화하는 방향.

---

## 오늘 정한 것 (결론)

1. **타깃 = POS 없이 운영되는 서비스업(3~10인).**
   미용실·네일·마사지·사진관·공방·카페·학원·카센터·세탁소·병원/한의원 등. 공통 고통: 고객·예약·매출·비용을 수기/카톡/엑셀로 분산, 급여·스케줄에 매달 2~10시간 낭비, 분석 자동화 전무, POS는 비용·난이도로 부담. → 경량 ERP+CRM+그룹웨어가 필요.

2. **5대 엔진 구조.**
   ① CRM(고객·이력·재방문·쿠폰·객단가) ② 예약·스케줄링(태블릿/웹 입력·직원 스케줄·캘린더) ③ 매출·정산(가격 자동계산·옵션/할인·집계·결제수단·피크타임) ④ 경비·급여(지출·영수증·근태·급여 자동계산·명세서 PDF·순수익) ⑤ 전자결재·업무관리(휴가·지출승인·공지·문서·권한).

3. **핵심 차별점 = 단골 관리 자동화(예약 알림·쿠폰).**
   예약 등록 → 알림톡/문자 자동 발송(방문 횟수·무료 쿠폰까지 남은 횟수·재방문 추천). 초기=메시지 복사→카톡 붙여넣기(무료), 상용=카카오 비즈메시지(알림톡) 정식 연동.

4. **BM = 구독 + 태블릿 대여.**
   월 1~5만원 구독형 + 선택적 태블릿 대여(월 9,900~29,000원 직원 수 기준 + 태블릿 3,000~7,000원). SaaS+하드웨어 결합으로 도입 장벽↓·월 정기 수익 안정.

5. **업종 중립 설계로 확장.**
   "고객 + 예약 + 매출 + 비용 + 직원" 구조가 있는 업종이면 치환 적용(미용실→시술, 사진관→촬영, 학원→출결, 카센터→수리, 병원→진료, 공방→수강/재료).

## 아직 안 정한 것 (다음에, 필요해지면)

- **추진 여부·시점**: 이건 역사 콘텐츠(북극성)와 별개 사업 아이디어 — 실제 착수 여부·우선순위 미정(기획 요약 단계).
- **후속 산출물**: 전체 화면 설계(Wireframe)·DB 테이블 구조·가격표 디자인·투자용 발표 슬라이드(10장)는 필요 시.

→ 별개 사업이라, 현재 초점(역사 콘텐츠)과의 우선순위 판단이 선행돼야 함.

## 확인된 구조 (이번에 OK 난 것)

MVP 우선순위:

| 단계 | 내용 |
|------|------|
| 1단계 (핵심) | 예약 입력·고객 DB·작업/수업 기록·매출 자동계산·월/일 리포트·근태/급여·경비/순수익·태블릿 UI |
| 2단계 (경쟁력) | 쿠폰 자동계산·예약 알림톡·시간대/요일 분석·직원별 매출·세무용 PDF/엑셀 |
| 3단계 (확장) | 업종별 UI 템플릿·정기결제·간단 재고관리 |

한 줄 요약: 수기·엑셀 소상공인의 예약·매출·경비·급여·고객관리를 한 번에 해결하는 경량 ERP+CRM+그룹웨어, 월 1~5만원 SaaS.
