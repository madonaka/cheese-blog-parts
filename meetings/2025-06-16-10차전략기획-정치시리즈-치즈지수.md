# 2025-06-16 회의 — 10차 전략 기획: 정치인+정당계보 콘텐츠 전략 고도화

> 참여: 치즈, 루멘
> 주제: 정당계보 기반 정치인 콘텐츠 시리즈 정식 론칭, 콘텐츠 생성 매뉴얼 정식화, 치즈지수 평가 체계 구축
> ※ 이관 노트. Notion "Cheese Lab 10차 전략 기획 회의"를 회의록 형식으로 옮긴 것. 치즈지수 rubric은 요약, 첨부 이미지 제외.

---

## 배경 (왜 이 회의를 했나)

기존 가계도 시리즈에서 벗어나, 현대 정치사 흐름(정치인+정당계보) 중심으로 콘텐츠 축을 전환하는 것을 검토했다. 수요는 미지수지만 정당계보를 자료화한 콘텐츠가 거의 없어 **공급 독점 가능성**이 크다는 점이 핵심 동기. 함께 정치인 전략가치를 수치화하는 평가 시스템(치즈지수)도 세웠다.

---

## 오늘 정한 것 (결론)

1. **"치즈랩 정치 시리즈" 정식 론칭.**
   정당계보 기반 정치인 콘텐츠를 정식 등록. 이해찬·유시민·김대중 3편 업로드 완료(초기 총 조회 ~10회). 차별점은 공급 희소성. 포맷: 정치인 생애를 정당 계보 흐름으로 추적하는 인포그래픽 서사, 티스토리 + 네이버 블로그 이중 업로드. "콘텐츠 누적 시 승부 가능"으로 판단.

2. **시리즈 10편 누적을 1차 전략 목표로 설정.**

3. **콘텐츠 생성 매뉴얼 10단계 정식화.**
   키워드사운드(황금키워드) → 나무위키·네이버 자료 → 정당계보 포맷 매칭 + 퍼플렉시티 검증 → Canva/PPT 시각자료 → GPTs 초안 → Claude 감수 → 티스토리 업로드(목차·버튼·링크) → GPTs로 블로그용 재작성 → Claude 감수 + SEO 제목 → 네이버 블로그 업로드.

4. **치즈지수 v1.0 확정 — 정치인 전략가치 평가지표.**
   6개 항목으로 점수화, 자동 평가 프롬프트 제작 완료. 정당 규모(100석↑ / 20~99석 / 1~19석 / 0석)별 차등 보정. 정책 '브랜드'는 제외하고 정책 실행력·설계력으로 대체.

5. **2단계 확장 전략 채택.**
   콘텐츠 10편 이상 누적 시 → 유튜브(10분 콘텐츠) / 전자책 / 스마트스토어 상품화로 확장.

## 아직 안 정한 것 (다음에, 필요해지면)

- **저작권 리스크 대응**: 실사 이미지 → 일러스트/스케치 기반 전환은 '향후 고려'(미확정). "나무위키 참조" 명시·사실 확인은 필수 원칙으로.
- **압축형 카드 콘텐츠 포맷**: 포스터형 / 캐러셀형 등 SNS 확산형은 탐색 단계.
- **이번 주 잔여 인물**: 6/20~23 인물 미정.
- **치즈지수 만점 수치**: 원문에 "최대 7.0점" 표기와 항목 합(요약표 기준 10.0)이 불일치 → 만점·환산(→5.0점) 기준 재정리 필요.
