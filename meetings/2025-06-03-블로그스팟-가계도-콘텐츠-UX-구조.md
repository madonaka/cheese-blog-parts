# 2025-06-03 회의 — 블로그스팟 가계도 콘텐츠 UX 구조 확정

> 주제: 티스토리 한계 극복을 위한 블로그스팟 콘텐츠 허브의 가계도 콘텐츠 UI/UX 흐름 설계
> ※ 이관 노트. Notion "블로그스팟 콘텐츠 UX 전략 보고서"(8차 전략 기획 회의 부속)를 회의록 형식으로 옮긴 것.

## 배경 (왜 이 회의를 했나)

기존 티스토리 중심 콘텐츠 구조는 유입-탐색-전환을 정돈하기 어렵다는 한계가 있었다. 이를 극복하려고 구글 블로그스팟 기반의 **홈페이지형 콘텐츠 허브**를 새로 착수했고, 그 첫 단계로 핵심 콘텐츠인 **가계도**의 UX 구조를 설계했다. 가계도는 단일 인물이 아니라 시기별 계보 흐름(예: 조선왕조계보 ①)을 단위로 하므로, 탐색성과 정보 명료도를 함께 잡는 다단계 흐름이 필요했다.

---

## 오늘 정한 것 (결론)

1. **3단 구조로 간다.**
   탭 기반 탐색 / 카드 기반 콘텐츠 구성 / 외부 티스토리 연결. 콘텐츠마다 '요약 → 심화 → 확장'으로 이어지는 내비게이션. 모바일 대응·시각적 명료성을 위해 단순·명확 UI를 유지.

2. **가계도 탐색 흐름 = 시대 → 시기 → 콘텐츠 카드 → 상세.**
   상단 고정 탭 [가계도 탐색] → '조선/고려/대한제국' 등 시대 버튼 → 아코디언으로 '조선초/중기/후기' 시기 버튼 펼침 → 해당 시기 콘텐츠 카드 리스트(조선왕조계보 ①②…) → 카드 클릭 시 상세 페이지 진입.

3. **콘텐츠 카드 구성 확정.**
   대표 썸네일 · 제목(예: 조선왕조계보 ①) · 간략 설명(또는 인물 하이라이트) · [자세히 보기] 버튼. 시리즈성은 '다음 콘텐츠 보기'로 자연스러운 이동.

4. **상세 페이지 레이아웃 = 상단 CTA 3 + 하단 인물 카드.**
   상단 고정 CTA 3개(📌 가계도 그림 / 🗓 해당 시대 연표 / 🏞 관련 역사 여행지) — 색상 대비·아이콘·고정 위치로 탐색력 확보. 하단에는 등장 핵심 인물 카드 그룹(사진·이름/직위·재위·대표 사건), 클릭 시 요약 팝업 → '자세히 보기'로 티스토리 이동. 카드는 2~3열, 모바일 1열 리플로우.

---

## 아직 안 정한 것 (다음에, 필요해지면)

- **티스토리 → 블로그스팟 복귀 동선**: 외부 콘텐츠에서 다시 허브로 돌아오게 티스토리 쪽 구성을 수정해야 함(방식 미정).
- **인물 사진 부재 시 대체안**: 대체 일러스트 / 컬러 박스 중 택1 미정.
- **카드 열 수 기준**: 2열 vs 3열 배치 기준 미확정(반응형 규칙만 있음).

→ 실제 HTML/CSS 구현 시안을 만들며 정한다.

## 확인된 구조 (이번에 OK 난 것)

| 항목 | 상태 | 방식 |
|------|------|------|
| 반응형 | OK | 기본 지원, 모바일에서도 카드/버튼 UX 유지(자동 1열) |
| 디자인 | OK | 블로그스팟 테마 기준, 색상 대비 + 정돈된 구조 |
| 인터랙션 | OK | 아코디언(접이식), 카드 클릭 시 팝업 or 펼침형 정보 |
| 연결 구조 | OK | 내부 요약 → 외부 티스토리로 새 창 이동 or 모달 |
| 탐색 단계 | OK | 시대 → 시기(아코디언) → 콘텐츠 카드 → 상세(CTA+인물카드) |
