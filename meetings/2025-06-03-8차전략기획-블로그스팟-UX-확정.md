# 2025-06-03 회의 — 8차 전략 기획: 블로그스팟 가계도 콘텐츠 UX 구조 확정

## 배경 (왜 이 회의를 했나)

기존 티스토리/네이버 블로그는 목록형 구조라 콘텐츠 탐색성이 떨어졌다. 이를 극복하려고 블로그스팟 기반 홈페이지형 **콘텐츠 탐색 허브**를 구축하기로 하고, 이번 회의에서 가계도 콘텐츠를 기준으로 UX 구조를 확정했다. (상세 설계 근거는 같은 날 블로그스팟 UX 전략 보고서에 정리.)

---

## 오늘 정한 것 (결론)

1. **블로그스팟 콘텐츠 UX = 시리즈/탭/카드 중심으로 확정.**
   사용자 접근 흐름을 `탭 → 시대/시기 → 콘텐츠 카드`로 단순화. 가독성과 유입률을 함께 확보하는 탐색 중심 메인 페이지 설계.

2. **가계도 4단 UX 구조 확정.**
   `콘텐츠 카드 → 인물 카드 그룹 → 정보 팝업 → 외부(티스토리) 연결`. 계층은 조선 → 조선초 → 콘텐츠 카드(조선왕조계보 ① 등) → 등장 인물 카드 그룹. 인물 카드 클릭 시 요약 정보 → 티스토리 심화 콘텐츠로.

3. **상단 고정 CTA 3버튼으로 유입 유도.**
   상세 페이지 상단에 📌 가계도 그림 / 🗓 연표 / 🏞 역사 여행지 3개를 고정. 티스토리·외부로 유입을 유도해, 블로그스팟을 **광고/구독 유도형 콘텐츠 허브**로 기능화한다.

4. **연표·답사 콘텐츠도 동일 구조로 확장 예정.**
   이번에 잡은 카드/CTA/계층 구조를 다른 콘텐츠 축에도 그대로 적용한다.

## 아직 안 정한 것 (다음에, 필요해지면)

- **콘텐츠별 인물 카드 구성·티스토리 링크 정리**: 실제 데이터 채우기 미착수.
- **HTML/CSS 실제 UI 시안 개발**: 구조만 확정, 구현 시안은 다음.
- **연표·답사 콘텐츠 동일 UX 적용**: 방향만 정함, 세부 설계는 다음 회의.

→ 인물 카드 기획과 실제 UI 시안 개발로 이어간다.
