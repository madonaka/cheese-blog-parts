# 2026-07-08 회의 — 지도·연표 결합의 "왜" (학습자 니즈)

> 참여: 치즈랩 대표, Claude
> 주제: T-013/T-014가 실제로 어떤 기능인지 확인하고, 학습자가 왜 이 결합을 필요로 하는지를 착수 전 UI 판단 기준으로 박제

---

## 배경 (왜 이 회의를 했나)

대표가 T-013을 "사건이 일어난 곳의 지도 좌표에 핀을 찍어주는 기능"으로 이해하고 있었다.
지도 뷰어 구조를 확인해 보니 실제 결합 단위가 그 이해와 달랐고(딥링크 파라미터 `?map=&y=&nation=` 에 위경도가 없음),
이 차이를 짚는 김에 "학습자가 이 기능을 대체 왜 필요로 하는가"를 정리해 T-013 착수 시 화면 판단의 기준으로 남기기로 했다.

---

## 오늘 정한 것 (결론)

1. **T-013/T-014는 "좌표 핀"이 아니라 "시대·나라 단위 연동"이다 — 대표의 첫 이해를 교정.**
   지도 뷰어는 점(사건 장소)이 아니라 **면(`territories`, 나라 영토 폴리곤)** 과 **시대(`years`, 스냅샷)** 로 움직인다.
   딥링크는 `map`(어느 지도)·`y`(연도 → 가장 가까운 시대 스냅샷으로 점프)·`nation`(그 사건 주체 나라를 하이라이트) 세 개이고, 사건의 lon/lat는 데이터에 없다.
   즉 "지도에서 보기" = 핀 표시가 아니라 **그 사건이 놓인 시대의 판도를 열고 해당 나라를 강조**하는 것.

2. **학습자가 느끼는 진짜 갈증 = 시간축과 공간축 중 하나만 붙잡아 나머지가 계속 비는 답답함.**
   T-013(연표→지도)은 시간을 붙잡은 학습자에게 공간을, T-014(지도→연표)는 공간을 붙잡은 학습자에게 시간을 채워준다. 이 한 줄이 두 티켓의 존재 이유다.

3. **학습자의 구체적 속마음을 UI 설계 기준으로 박제한다.**
   - "그래서 이게 어디야?" — 낯선 지명(북벌의 광저우→우한→난징→베이징 등)에서 방향·거리 감각이 끊김.
   - **"그때 세계는 지금과 다르게 생겼겠지?"** — 학습자는 무의식적으로 현재 지도를 떠올리는데, 오스만·소련·분단 전 국경은 현재 지도로 상상하면 다 틀린다. 이 오해 교정이 시대 스냅샷+나라 하이라이트가 주는 핵심 값. (다섯 물음 중 가장 중요)
   - "이 나라, 이때 얼마나 컸어?" — 텍스트의 "영토 확장"은 크기·인접 관계 감각이 없다. 면을 봐야 온다.
   - "이 판도일 때 무슨 일이 벌어졌지?" (T-014) — 지도(결과)에서 사건(원인)으로 넘어가려는 욕구.
   - "같은 연도에 다른 데선 뭐 했어?" (T-014) — 미국·소련·중국을 같은 시간대의 거울 시리즈로 쓰는 콘텐츠 구조가 여기서 살아남.

4. **사건 좌표 핀은 지금 로드맵에 없는 별개 심화 기능으로 분리한다.**
   학습자의 1차 갈증은 "정확히 이 좌표에서 무슨 일"이 아니라 "이 시대에 이 세력이 이만큼"이라, 점보다 면·시대 연동이 먼저다.
   좌표 핀은 특정 전투·조약을 콕 짚어 보려는 심화 학습자용이며, 사건 데이터에 lon/lat를 추가하고 뷰어에 마커 레이어를 새로 얹어야 해서 결합 4단계보다도 더 나간 확장이다.

## 아직 안 정한 것 (다음에, 필요해지면)

- 사건 좌표 핀(마커 레이어)을 결합 5단계 티켓으로 발행할지 — 필요를 느끼는 심화 학습자 신호가 실제로 보일 때 판단.
- 위 다섯 물음 중 T-013 1단계 UI에 어디까지 반영할지(딥링크+나라 하이라이트만인지, 연도 근사 안내 문구까지인지).
- T-013 착수 시점과 현재 `doing`인 T-006(연표 고도화) 처리(전환/병행).

→ T-013 착수 지시가 나올 때 위 세 가지를 함께 결정한다.

## 확인된 구조 (이번에 OK 난 것)

| 항목 | 상태 | 방식 |
|------|------|------|
| 결합 키 | OK | 사건 좌표가 아니라 **연도 + 나라(faction)** — 지도 나라명=learn_characters faction 표기(사전 준비 완료) |
| 지도 표현 | OK | 핀/마커 없음. 시대 스냅샷 이동(`y`) + 나라 영토 강조(`nation`) |
| T-013 방향 | OK | 연표→지도 (시간을 붙잡은 학습자에게 공간을) |
| T-014 방향 | OK | 지도→연표 (공간을 붙잡은 학습자에게 시간을) |
| 좌표 핀 | 보류(별개) | 로드맵 밖 심화 기능 — 필요 시 5단계로 |
