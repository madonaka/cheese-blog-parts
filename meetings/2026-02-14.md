## 배경 (왜 이 회의를 했나)

네이버 블로그에 고화질 자료를 올리면 무단 도용·복제가 잦았다.
그래서 현재 구글 블로그에는 저장 방지 코드 + 저화질 자료만 올리는 중인데,
유출은 막았지만 정작 고품질 자료를 돈으로 바꿀 기회를 통째로 버리고 있었다.
→ 기존 자료를 유료 콘텐츠로 전환해 직접 판매하고, 외부 플랫폼 수수료도 줄이자는 게 출발점.

---

## 오늘 정한 것 (결론)

1. **Blogger 기반 토스페이먼츠(Toss Payments) 연동 PG 시스템을 구축한다.**
   블로그(Blogger)에 결제창을 붙여, 자료를 그 자리에서 결제·구매하도록 만든다.

2. **결제/다운로드/판매기록을 Apps Script + Google Sheets로 자동화한다.**
   별도 서버 없이 기존 Google Workspace 인프라로 결제 처리와 판매 기록을 자동으로 남긴다.

3. **핵심 가치 = 수수료 절감 + 저작권 보호.**
   외부 플랫폼에 내던 수수료를 줄이고, 직접 판매로 콘텐츠 통제권과 저작권 보호를 강화한다.

4. **안전한 다운로드 제공 + 최소 운영 관리 기능만 붙인다.**
   과설계하지 않고, 구매자가 안전하게 받을 수 있는 다운로드와 최소한의 운영 화면부터.

## 아직 안 정한 것 (다음에, 필요해지면)

- 가격 정책 (개별 판매 / 묶음 / 할인)
- 다운로드 권한 부여·회수 정책의 세부 (환불 시 회수 등)
- 결제 실패·중복 결제 예외 처리 방식

→ MVP로 결제→다운로드 한 흐름을 먼저 돌려본 뒤, 실제 판매 데이터를 근거로 정책을 확정.
