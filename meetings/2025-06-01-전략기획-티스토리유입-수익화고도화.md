# 2025-06-01 회의 — 7차 전략 기획: 티스토리 유입 전략 + 수익화 실험 고도화

> 참여: 치즈, 루멘
> 주제: 가계도 이관 마무리, 광고·CTA 최적화, 마케팅 기법 학습(아로스TV), 일본어 루틴, 가계도 GPTs
> ※ 이관 노트. Notion "Cheese Lab 7차 전략 기획 회의"를 회의록 형식으로 옮긴 것. 안건 5개를 6개 섹션으로 재배치.

---

## 배경 (왜 이 회의를 했나)

가계도 콘텐츠 이관을 이어가면서, 유입 전략과 블로그 수익화 실험을 한 단계 끌어올리는 자리. 광고 배치·CTA 문구를 실사례(아로스TV)로 다듬고, 콘텐츠 생산 효율을 위한 GPTs 도입과 일본어 루틴 착수도 함께 다뤘다.

---

## 오늘 정한 것 (결론)

1. **가계도 남은 5편, 6월 1주차 내 이관 마무리.**
   12편 중 7편 완료. 남은 5편 순서: 효종&현종 → 숙종&경종 → 영조&정조 → 순조~철종 → 고종&순종. 모바일에서 가계도 그림이 작게 보이는 점을 역이용해 **그림에 티스토리 링크를 삽입**, 클릭해야 크게 보는 구조로 강제 유입 동선을 만든다.

2. **광고 = 수동 배치 + 행동 유도형 CTA로 고도화.**
   광고는 문단 끝·주제별 정리 직후에 배치(거부감↓·클릭률↑). 자동광고는 쓰되 중요한 위치엔 수동 광고 중심. CTA는 "알아보기" 대신 궁금증을 유발하는 행동 유도형 문장으로. 수익 즉시 반응 확인(전일 0.06 → 당일 0.16달러).

3. **마케팅 기법 학습·내재화 착수.**
   아로스TV 등 블로그 수익화 사례에서 전략 키워드 정리: 거미줄 전략(내부 링크 강화)·3버튼 전략(CTA 시각 구분·반복)·외부 유입 다변화(지식인·카페·페북·스레드)·파소나 구조(Pain→Agitate→Solution→Offer→Narrow→Action). 6월 1주차 내 마케팅 책 1권 선정 + 적용 사례 콘텐츠 제작 목표.

4. **일본어 학습 루틴 설계 착수.**
   전략기획서 기준 우선순위 1위 단기 목표인데 실제 루틴은 미수립 상태. 이번 주 내 회화 중심(매일 청취+회화+로그) 기초 계획을 세워 주간 실행 시트에 반영.

5. **가계도 콘텐츠 생성 GPTs 도입.**
   자동 생성 툴 제작 완료. 워크플로: 주제 입력 → 자료 입력(나무위키·위키백과) → GPT가 A/B/C 서술 흐름 제시 → 선택 → 1,500~1,700자 자동 서술 → 일부 직접 수정해 최종화. 효과: 글쓰기 시간 단축·생산 균일화·스타일 통제. 별도 결정 없이 조금씩 보완해 사용.
