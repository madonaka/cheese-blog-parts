# cheese-blog-parts
치즈의 역사블로그 부품화

Blogger / Tistory / 일반 웹페이지에서 사용할 수 있는
Cheese Lab 전용 UI 컴포넌트 모음입니다.

## 구성

- `lightbox/` : 이미지 라이트박스 (줌/드래그)
- `footnote/` : 위키 스타일 주석 툴팁 + 모바일 모달
- (예정) `quiz/` : 객관식/서술형 연습문제
- (예정) `components/` : 인포박스, 배지, 시대 연표 등 CSS 컴포넌트

각 폴더 안의 `README.md`에 개별 사용법과 Blogger 코드 스니펫이 정리되어 있습니다.
