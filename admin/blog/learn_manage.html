숫자로 시작하는 제목(예: 1. 제목)은 자동으로 브라운/골드 제목 스타일이 적용되며, 화살표(→)가 포함된 단락은 자동으로 단계별 베이지 박스 스타일이 적용됩니다.

청크 개요

퀴즈 대량 입력 (CSV)

[CSV 입력 형식 가이드]
첫 줄에 헤더(유형,문제,정답,보기A,보기B,힌트,해설,연계세트ID)를 적고 줄바꿈하여 데이터를 입력하세요.
- 유형 [필수]: ox (OX 퀴즈) 또는 short (이지선다형)
- 문제 [필수]: 퀴즈 질문 내용
- 정답 [필수]: OX는 O 또는 X / 이지선다는 보기A 또는 보기B에 적은 텍스트 값과 정확히 일치해야 함
- 보기A, 보기B [이지선다형 필수]: 이지선다형 선택지 텍스트 (OX 퀴즈인 경우 비워둠)
- 힌트, 해설, 연계세트ID [선택]: 생략 가능하며, 생략하더라도 쉼표(,) 구분자는 유지해야 함
