필드명 타입 설명 / 예시 필수
name String 대표 이름 (예: "장쩌민") 필수
type String 구분 (`"character"` 인물, `"event"` 사건, `"concept"` 사상/개념, `"nation"` 나라/국가) 필수
alt_names Array 별명, 이명, 한자 및 번역 동의어 배열 선택
birth_year String 출생연도 또는 사건 발생시기 (예: `"1889"`, `"1989년"`) 선택
death_year String 사망연도 (인물=몰년, 나라=멸망 연도. 예: `"1927"`, `"1991"`. 현존 시 비움) 선택
terms Array 집권 기간 배열 `[{start, end}]` (지도자, 생몰 연표에서 빨간 구간으로 표시 · 재집권 시 여러 개, end 비우면 재임 중) 선택
term_start / term_end String 레거시 호환 필드 (terms[0]의 값이 자동 기록됨) 선택
faction String 국가/진영 (인물=지도자만 보기 그룹 구분, 사건=세로 연표의 나라 컬럼 분류. 나라(nation) 항목명과 일치시키면 연결됨. 예: `"소련"`, `"중국"`) 선택
lineage String 계보/역사권 (나라(nation) 전용. 같은 역사로 묶을 나라들을 인접 정렬·그룹화. 예: `"러시아사"` → 러시아제국·소련) 선택
is_global Boolean 세계 공통 사건 여부 (사건(event) 전용. `true`면 세로 연표에서 전체 폭 가로 점선으로 표시. 예: 세계대전) 선택
memo String 해당 항목에 대한 세부 메모 및 역사적 설명 선택
related_events Array 연관된 사건의 name 목록 (예: `["5.4 운동", "국공합작"]`) 선택
appearances Array 등장 학습 위치 배열. 하위 객체 필드:
- series_name: 시리즈명 (선택)
- title: 학습단계(편) 제목 (필수)
- content_connection: 연결 설명/메모 (선택)		 선택
