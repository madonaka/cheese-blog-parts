factions Array 국가/진영 배열 (사건(event) 전용 — 세로 연표 나라 컬럼 분류. 여러 나라가 동시에 관련된 사건이면 모두 나열하면 각 나라 컬럼에 함께 표시됨. 예: `["중국", "미국"]`) 선택
lineage String 계보/역사권 (나라(nation) 전용. 같은 역사로 묶을 나라들을 인접 정렬·그룹화. 예: `"러시아사"` → 러시아제국·소련) 선택

appearances Array 등장 학습 위치 배열. 하위 객체 필드:
