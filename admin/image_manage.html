이미지 40% / 50% / 60% 미리보기 (클릭 시 새 창)

업로드 전에 브라우저에서 리사이즈 + (선택) 블러 + (선택) 워터마크(오버레이) + (선택) 문구를 적용합니다. 미리보기 이미지를 클릭하면 새 창(탭)에서 확인할 수 있습니다.

원본 이미지
워터마크 PNG
JPEG 품질
0.72 낮을수록 용량↓ / 글자 뭉개짐↑
미세 블러(px)
0.5 0.4~0.6부터 권장
워터마크 투명도
0.25 낮을수록 약하게 보임 (1.00 = 워터마크 원본 그대로)
표시 문구
자동으로 “VYYYYMMDD” + “원본 대비 % / 블러 px”가 함께 붙습니다.
글자 크기
18px 배경 없음(투명) / 글자 검정
업로드 선택
칩(40/50/60)을 눌러 업로드 대상을 바꿀 수 있어요.
업로드가 성공하면 아래에 “HTML 스니펫”이 나오고, 바로 복사할 수 있습니다.

업로드 연결(현재 활성)

현재 파일은 “미리보기 생성 + 새 창 확인 + 선택 업로드 + DB 저장 + HTML 스니펫 생성”까지 포함합니다. Apps Script 쪽에서는 mode=upload(JSON)로 받아 Drive 저장 + 시트 기록을 수행해야 합니다.
