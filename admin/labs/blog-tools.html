티스토리/블로그스팟 HTML에 그대로 붙여 넣는 스니펫을 생성합니다.
Flashcard Drag Order Quiz v0.7
플래시 카드
드래그 순서 퀴즈
입력
빠른 입력: 아래 “일괄 붙여넣기”에 앞면[TAB]뒷면 또는 앞면|뒷면 형식으로 한 줄에 1개씩 붙여넣고 “불러오기”를 누르세요.
일반 입력(행 추가)도 그대로 사용 가능합니다.
출력
- 카드 내용에 < > 같은 문자가 있어도 깨지지 않도록 내부에서 escape 처리합니다.
- 스니펫에 </script>가 “그대로” 들어가면 브라우저가 중간에서 스크립트를 끊어버릴 수 있어서, 안전하게 분할해서 출력합니다.
입력
정답 입력을 쉽게: 기본은 입력한 순서가 정답입니다. (예: 1,2,3,4...)
그리고 보기 배치만 랜덤(섞기)할 수 있어요. → 숫자 정답을 거의 안 쓰게 됩니다.
위 체크를 끄면 입력한 순서(1,2,3...)가 자동 정답이라 여기 입력할 필요가 없습니다.
출력
- 보기 섞기 ON이면, 정답은 그대로 두고 보기만 랜덤으로 배치됩니다. (입력 순서 = 정답을 유지)
Tips: 블로그스팟은 “테마 HTML”에 붙여넣을 때 스크립트/스타일 위치에 따라 동작이 달라질 수 있어요. 가능하면 게시글 HTML(또는 HTML/자바스크립트 가젯)에 붙여넣는 방식이 가장 안정적입니다.
